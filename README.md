## :rocket: 体型管理アプリ

写真やカレンダーを使って体型写真を管理するアプリです。 以下を登録できます。

- 測定時間
- 体重
- 体脂肪率
- 画像

データは全て端末内のDBに保存します。

## :blue_book: 使い方

AndoirdStudioでクローンすれば動きます。
